# DevEx Calculator 💰

## About
DevEx Calculator is a fast, lightweight web tool that helps Roblox developers convert Robux into USD using the latest DevEx rate (March 2026 update).

It is built as a single-page application optimized for GitHub Pages with a clean Neumorphism (Soft UI) design.

---

## Key Features
- ⚡ Instant Robux to USD conversion
- 🎯 Based on latest DevEx rate ($0.0038)
- 🎨 Modern Neumorphism UI
- 📱 Fully mobile responsive
- 🚀 Lightweight & fast loading
- 🔒 No login required

---

## Tech Stack
- HTML5
- CSS3 (Neumorphism Design)
- Vanilla JavaScript

1. Enter your Robux amount
